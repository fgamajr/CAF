# 07_MODELOS_TCU

**Modelos e templates TCU. Padrões de formato, estrutura e apresentação para relatórios de auditoria.**

## Propósito

Contém templates e exemplos de documentos aprovados pelo TCU. Use como referência para formato, estrutura narrativa e apresentação de resultados.

## Conteúdo

### exemplo_instrucao_processo.pdf
- **Tipo**: Exemplo completo de processo de auditoria
- **Uso**: Referência de como organizar e apresentar processo
- **Aplicação**: Consulte para estrutura geral de peças e tramitação
- **Qualidade**: Modelo aprovado pelo TCU

### template_relatorio_auditoria.docx
- **Tipo**: Template de relatório de auditoria operacional
- **Uso**: Guia de seções, formatação e estrutura
- **Aplicação**: Siga para introdução, achados, conclusões
- **Qualidade**: Template oficial TCU

## Como Usar

### Para Redatores
1. Abra template_relatorio_auditoria.docx como referência
2. Observe estrutura de seções (Introdução, Achados, Conclusão)
3. Mantenha formatação e estilo em `01_RELATORIO_V2/`
4. Siga padrão de numeração e referências cruzadas

### Para Validadores
1. Confronte estrutura contra exemplo_instrucao_processo.pdf
2. Verifique conformidade com template oficial
3. Confirme que peças estão ordenadas conforme padrão TCU
4. Valide formatação final

## Padrões Observar

- Margens, fontes, espaçamento conforme template
- Numeração sequencial de peças
- Referências cruzadas explícitas
- Sumário e índices conforme padrão
- Assinaturas e responsabilidades formais

## Aviso

Estes são MODELOS e EXEMPLOS. A auditoria TC 011.073/2025-0 pode ter variações conforme instruções específicas do processo. Consulte `02_FONTE_VERDADE/` para instruções específicas desta auditoria.
