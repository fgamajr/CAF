## Fluxo de Trabalho

### Para Redatores
1. Consulte `02_FONTE_VERDADE/` para matriz de achados (PEÇA 170) - esta é a "constituição" inviolável
2. Revise `02_FONTE_VERDADE/Anexo_Comentarios_Gestor.md` para tom e calibração esperados
3. Redija/revise os documentos desta pasta com foco em consistência com a matriz
4. Use `04_PECAS_EVIDENCIA/` como fonte de fatos e evidências

### Para Validadores
1. Confronte cada seção contra PEÇA 170 (Matriz de Achados)
2. Verifique alinhamento de tom com PEÇA 169 (Anexo Comentários Gestor)
3. Valide citações e referências a evidências em `04_PECAS_EVIDENCIA/`
